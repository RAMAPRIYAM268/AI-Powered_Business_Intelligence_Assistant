# AI-Powered Business Intelligence Assistant

> An intelligent Business Intelligence solution developed using **n8n, MySQL, Power BI, and Generative AI** to enable natural language interaction with business data. The project automates SQL query generation, data retrieval, and dashboard reporting, allowing users to access business insights without requiring technical expertise.

---

# Project Overview

Business users often rely on technical teams to retrieve data and generate reports because they lack SQL knowledge or experience with Business Intelligence tools. This dependency can delay decision-making and reduce operational efficiency.

This project presents an AI-powered Business Intelligence Assistant that allows users to ask business questions in natural language. Using Generative AI, the assistant automatically converts user queries into SQL statements, retrieves data from a MySQL database, and presents analytical insights through Power BI dashboards. The workflow is fully automated using n8n, providing a seamless and scalable reporting solution.
